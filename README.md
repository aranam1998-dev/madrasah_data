# Madrasah Data & Assets
এটি *Madrasah Management* অ্যাপ্লিকেশনের জন্য একটি পাবলিক রিসোর্স রিপোজিটরি। অ্যাপের ব্যবহারকারীরা যাতে খুব সহজে ডাটাবেজ এবং কুরআনের প্রয়োজনীয় ফাইলগুলো ডাউনলোড করতে পারে, সেজন্য এটি তৈরি করা হয়েছে।



### 📂 এখানে যা যা রয়েছে:

*   *database/*: কুরআন, হজ্জ গাইড, দু'আ এবং জিকিরের প্রয়োজনীয় SQLite (.db) ফাইলসমূহ।

*   *quran_pages/*: হাফেজী এবং তাজবীদ কুরআনের পাতার উচ্চমানের ছবিসমূহ।



### 🔗 ব্যবহারের নিয়ম (Base URL):

অ্যাপের ভেতর থেকে ফাইলগুলো অ্যাক্সেস করতে নিচের লিঙ্কটি ব্যবহার করা হয়:

https://raw.githubusercontent.com/aranam1998-dev/madrasah_data/main/



### 🛠 আমাদের লক্ষ্য:

এই রিপোজিটরি ব্যবহারের মূল উদ্দেশ্য হলো অ্যাপের সাইজ ছোট রাখা এবং ব্যবহারকারীকে প্রয়োজন অনুযায়ী নির্দিষ্ট ডাটা ডাউনলোডের সুবিধা প্রদান করা। মূল সোর্স কোডটি নিরাপত্তার স্বার্থে আমাদের প্রাইভেট রিপোজিটরিতে সংরক্ষিত আছে।
